## Создание и управление инфраструктурой для Yandex Cloud

- Все задачи rake
```bash
rake -T 
```
- Валидация terraform
```bash
rake test:terraform
```
- Развертывание инфраструктуры
```bash
rake tfa
```
- Деплой чартов
```bash
rake install_charts
```
- Удаление инфраструктуры
```bash
rake tfd
```
